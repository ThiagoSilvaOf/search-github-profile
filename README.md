# 🚀 Teste Técnico AVANTI

Este projeto é uma aplicação de busca de perfis do GitHub. Desenvolvido como parte do processo seletivo da Avanti.

## 🔧 Tecnologias usadas
- React
- CSS
- GitHub API

## 💻 Funcionalidades
- Buscar qualquer usuário do GitHub
- Exibir avatar, nome e bio
- Mensagem de erro personalizada
